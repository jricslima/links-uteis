# 🔗 Links Úteis - Dashboard de Produtividade v1.5

**Links Úteis** é uma central de ferramentas online e acessos rápidos desenvolvida para otimizar o fluxo de trabalho administrativo e técnico. Originalmente focado nas necessidades do Ministério da Saúde, o projeto evoluiu para uma interface universal, leve e totalmente customizável.

---

## 🚀 Funcionalidades

O Dashboard adapta-se ao seu estilo de trabalho com recursos inteligentes integrados em uma única página:

- **🌓 Dual-Mode Automático:** Alternância entre Modo Claro e Modo Escuro com salvamento de preferência via LocalStorage.
- **🔍 Busca Inteligente:** Filtro de ferramentas em tempo real com destaque (highlight) de texto e suporte a caracteres especiais/acentos.
- **🌡️ Dashboard em Tempo Real:** Relógio digital, data completa e integração com API de clima (Open-Meteo) configurada para Realengo, RJ.
- **📊 Contador Dinâmico:** Monitoramento automático do total de ferramentas cadastradas e contador de visitas global.
- **⚓ Navegação por Âncoras:** Acesso instantâneo às categorias (MS, Governo, PDF, Planilhas e Extras) com rolagem suave (Smooth Scroll).

---

## 📋 Categorias Integradas

O toolkit organiza as ferramentas essenciais para o dia a dia:

- **🏢 Sites MS:** Acesso direto ao SEI, Outlook, Petrvs, SouGov e sistemas do DATASUS.
- **🇧🇷 Governo:** Atalhos estratégicos para AGU Sapiens, Receita Federal e Jusbrasil.
- **📄 Ferramentas PDF:** Suíte completa para edição, conversão e OCR de documentos.
- **📊 Planilhas & Consultas:** Geradores de etiquetas, consultas CCTC e bases de dados do Google Drive.
- **📦 Extras:** Serviços de busca de CEP, Rastreamento de Objetos e consulta de operadoras.

## 🖥️ Instalação e Uso

### 🌐 Uso Online (GitHub Pages)
Basta acessar o link do projeto:
`https://jricslima.github.io/links-uteis/`

### 📂 Uso Offline (Modo Local)
1. Baixe o repositório ou o arquivo `index.html`.
2. Certifique-se de manter os ícones na mesma pasta do arquivo HTML.
3. Abra o `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge ou Brave).

---

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3:** Estrutura semântica com variáveis nativas (CSS Variables).
- **Vanilla JavaScript:** Lógica pura para manipulação de DOM e consumo de API, sem dependências externas pesadas.
- **Open-Meteo API:** Dados meteorológicos de alta precisão sem necessidade de chaves de API.
- **Shields.io:** Badges dinâmicos para estatísticas e status do projeto.
