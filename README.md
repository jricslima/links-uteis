# 🔗 Links Úteis - Dashboard de Produtividade v1.5 (Edição Realengo)

**Links Úteis** é uma central de ferramentas online e acessos rápidos desenvolvida para otimizar o fluxo de trabalho administrativo e técnico. [cite_start]Originalmente focado nas necessidades do Ministério da Saúde, o projeto evoluiu para uma interface universal, leve e totalmente customizável. [cite: 46, 47, 48]

---

## 🚀 Funcionalidades de Elite

[cite_start]O Dashboard adapta-se ao seu estilo de trabalho com recursos inteligentes integrados em uma única página: [cite: 50, 51]

- [cite_start]**🌓 Dual-Mode Automático:** Alternância entre Modo Claro e Modo Escuro com salvamento de preferência via LocalStorage. [cite: 52]
- [cite_start]**🔍 Busca Inteligente:** Filtro de ferramentas em tempo real com destaque (highlight) de texto e suporte a caracteres especiais/acentos. [cite: 53]
- [cite_start]**🌡️ Dashboard em Tempo Real:** Relógio digital, data completa e integração com API de clima (Open-Meteo) configurada para Realengo, RJ. [cite: 54]
- [cite_start]**📊 Contador Dinâmico:** Monitoramento automático do total de ferramentas cadastradas e contador de visitas global. [cite: 55]
- [cite_start]**⚓ Navegação por Âncoras:** Acesso instantâneo às categorias (MS, Governo, PDF, Planilhas e Extras) com rolagem suave (Smooth Scroll). [cite: 56]

## 📋 Categorias Integradas

[cite_start]O toolkit organiza as ferramentas essenciais para o dia a dia: [cite: 58, 59]

- [cite_start]**🏢 Sites MS:** Acesso direto ao SEI, Outlook, Petrvs, SouGov e sistemas do DATASUS. [cite: 60]
- [cite_start]**🇧🇷 Governo:** Atalhos estratégicos para AGU Sapiens, Receita Federal e Jusbrasil. [cite: 61]
- [cite_start]**📄 Ferramentas PDF:** Suíte completa para edição, conversão e OCR de documentos. [cite: 62]
- [cite_start]**📊 Planilhas & Consultas:** Geradores de etiquetas, consultas CCTC e bases de dados do Google Drive. [cite: 63]
- [cite_start]**📦 Extras:** Serviços de busca de CEP, Rastreamento de Objetos e consulta de operadoras. [cite: 64]

### [cite_start]🌐 Uso Online (GitHub Pages) [cite: 67]

[cite_start]`https://jricslima.github.io/links-uteis/` [cite: 69]

### [cite_start]📂 Uso Offline (Modo Local) [cite: 70]
1. [cite_start]Baixe o repositório ou o arquivo `index.html`. [cite: 71]
2. [cite_start]Certifique-se de manter os ícones na mesma pasta do arquivo HTML. [cite: 72]
3. [cite_start]Abra o `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge ou Brave). [cite: 73]

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3:** Estrutura semântica com variáveis nativas (CSS Variables). [cite: 77]
- **Vanilla JavaScript:** Lógica pura para manipulação de DOM e consumo de API, sem dependências externas pesadas. [cite: 78]
- **Open-Meteo API:** Dados meteorológicos de alta precisão sem necessidade de chaves de API. [cite: 79]
- **Shields.io:** Badges dinâmicos para estatísticas e status do projeto. [cite: 80]
